Maven

Maven is a build automation tool used primarily for Java projects. The word maven means ‘accumulator of knowledge’ in Yiddish.

WikipediaApache Maven

Maven wordt gebruikt om het leven van de programmeur te vergemakkelijken. In een XML-bestand (pom.xml) wordt informatie gespecifieerd over het project (naam, modules, build order, …) en benodigde projecten (al dan niet) van derden (dependencies).
